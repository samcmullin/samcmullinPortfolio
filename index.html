It's Cocktail Time!

JQUERY | JAVASCRIPT | SASS | COCKTAIL API

Looking for a random cocktail? It's Cocktail Time was built in collaboration with a partner and an API/Ajax call is made to a cocktail database based on the user's selection of alcohol type. Once the API call is complete, the drinks are stored in an array of objects, and a random drink will be displayed to the user based on their alcohol choice!
